## ğŸš€ Getting Started

1. Clone the repository:
```bash
git clone https://github.com/DRTX2/portfolio.git
cd portfolio
```

2. Install dependencies (for SCSS compilation):
```bash
npm install -g sass
```

3. Compile SCSS (if making changes):
```bash
sass ./scss/main.scss css/main.css
```

4. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```
